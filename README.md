# ticTacToe
A simple two player Tic Tac Toe game made with C++

## Setup
1. Clone the repository.
2. `cd` into the directory.
3. Run the following commands:
   ```sh
   # To compile the program:
   $ gcc main.cpp -o ticTacToe
   
   # To run the program:
   $ ./ticTacToe
   ```
